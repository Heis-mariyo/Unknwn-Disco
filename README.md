# UNKNWN DISCO - Website for the Talented

This project is a complete 6-page, fully responsive website built from a Figma design for the fictional creative agency "UNKNWN DISCO." It's built with a modern tech stack and features extensive, high-fidelity animations and interactions.

## üöÄ Key Features

* **6-Page Website:** Includes Home, About, Services, Creators, Join, and Contact pages.
* **Fully Responsive:** Mobile-first design that adapts to all screen sizes, from phones to desktops.
* **Shared Layout:** Uses Next.js App Router layouts for a persistent Header, Footer, and Floating Buttons.
* **Rich Animations:**
    * **Framer Motion:** Used for staggered fade-in animations on page load and grid filtering.
    * **Gradient Drift:** Headline text gradients slowly shift, giving a "live" feel.
    * **Neon Glow:** Subtle "neon shimmer" on headlines and hover-glow effects on the logo and buttons.
    * **Text Scramble:** A "hacker-style" text decode effect on nav links and headlines.
* **Interactive Components:**
    * **Filterable Grid:** A client-side, animated grid for the "Creators" page.
    * **Interactive Forms:** Custom-styled forms on the "Join" and "Contact" pages.
    * **Micro-interactions:** Detailed hover effects on all buttons, cards, and links.
